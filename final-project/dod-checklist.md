# Final Project "Definition of Done" Checklist

A student is ready for Session 10 graduation if they can check these 5 boxes:

[ ] Live URL: Site is accessible via a .vercel.app or .netlify.app link.  
[ ] Multi-Page: Navigation works between at least 3 distinct HTML files.  
[ ] Dynamic Feed: Data is being pulled from an external JSON or API.  
[ ] Closed Issues: Their GitHub Issue board shows completed/closed tickets.  
[ ] Valid Form: The contact form prevents invalid submissions via JavaScript.
